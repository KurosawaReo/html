メンバ 返り値 static 説明
GetWindowRect() INT_RECT 〇 ウィンドウサイズを四角形で取得。
GetWindowX() int 〇 ウィンドウの横幅を取得。
GetWindowY() int 〇 ウィンドウの縦幅を取得。
GetFps() int 〇 設定したfpsを取得。
InitDx(int windowWid, int windowHei, bool isWindowMode, int fps, bool isVSync = TRUE) ResultInt 〇 DxLibの初期化を行う。
LoopDx() void 〇 DxLibのループ処理を行う。
Init() void 〇 自動で呼ばれる。初期化処理を入れるところ。
Update() void 〇 自動で呼ばれる。更新処理を入れるところ。
Draw() void 〇 自動で呼ばれる。描画処理を入れるところ。
Quit() void 〇 ループが一巡し終わった後、ループ処理を抜け出す。
